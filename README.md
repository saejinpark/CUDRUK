# CODRUK ☕️

> 커피로 꺼드럭대고 싶을 때 꺼내는 앱

## 소개

커피 지식을 빠르게 훑고 싶을 때, 카페 가기 전에, 원두 고르기 전에, 친구한테 커피 얘기하기 전에.  
필요한 순간에 꺼내서 보는 커피 학습 앱입니다.

## 기능

- 📚 14개 챕터, 128개 커피 카드
- 🔍 입문 / 중급 / 고급 레벨 필터
- 📝 카드별 요약 + 상세 보기
- ❓ 퀴즈로 자기 검증
- 🔖 북마크 및 완료 체크
- 📊 챕터별 학습 진행도

## 기술 스택

- SwiftUI
- SwiftData
- iOS 17+

## 개인정보 처리방침

CODRUK은 사용자의 개인정보를 수집하지 않습니다.
모든 데이터(학습 진행도, 북마크 등)는 사용자의 기기에만 저장되며 외부로 전송되지 않습니다.
